HTML best practices for beginners

Here are 10 HTML best practices to keep in mind when working on your next project.

  1. Use proper document structure
  2. Declare the correct doctype
  3. Always close tags
  4. Don't use inline styles
  5. Use alt attribute with images
  6. Validate frequently Validate Now!
  7. Place external style sheets within the head tag
  8. Use meaningful tags HTML elements reference(MDN Web Docs)
  9. Use lowercase markup
  10. Reduce the number of elements on a page or make it DRY(DONT REPEAT YOURSELF)

HTML entities

What are HTML entities?

An HTML entity is a piece of text ("string") that begins with an ampersand ( & ) and ends with a semicolon ( ; ) . Entities are frequently used to display reserved characters (which would otherwise be interpreted as HTML code), and invisible characters (like non-breaking spaces). Learn More!
