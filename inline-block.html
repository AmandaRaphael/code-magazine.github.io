Block,Inline and Inline-block elements

Block level element Inline element Inline-block element
Block level elements always starts on a new line An inline element doesn‚Äôt require a new line to start. It can work within an existing line of content. Elements do not start on a new line.
Block level element can have customized margin and padding Top & bottom margin can not be applied to inline elements (Unless they are converted to block level element via CSS) Elements can have vertical and horizontal padding on all sides.
If the width is not manually set, A block level element always takes up full width of its container.f the height isn‚Äôt manually set, Block level element‚Äôs height is adjusted automatically to fit all the content and other elements within itself. An Inline element just takes up the width and height it needs. Elements accept width and height properties.
Block level element can contain other block level element or Inline-level element Inline-level elements can only contain data or other inline level elements. There will often be a small gap between inline-block elements.
